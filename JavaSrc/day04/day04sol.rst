집 색칠
=========================

점화식:

.. math::  

    D[n][i] =     
    \begin{cases}            
        col[1][i],                & {n = 1, i = 1, 2, 3}   \\[2ex]       
        \min (D[n-1][2], D[n-1][3]) + col[n][1],            & { n \ge 2, i = 1}   \\[2ex]
        \min (D[n-1][1], D[n-1][3]) + col[n][2],            & { n \ge 2, i = 2}   \\[2ex]         
        \min (D[n-1][1], D[n-1][2]) + col[n][3],            & { n \ge 2, i = 3}   \\[2ex]         
    \end{cases}
